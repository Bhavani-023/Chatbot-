# Chatbot
# Voice Chat Bot using Python 🎙️🤖

## 📌 Project Description

This project is a **simple Voice Chat Bot built using Python**.
The chatbot listens to the user's voice commands through the microphone, processes the command using speech recognition, and responds with voice output.

It can perform basic tasks such as greeting the user and opening websites like **YouTube** and **Instagram**.

---

## 🚀 Features

* 🎤 Voice command recognition
* 🔊 Text-to-speech responses
* 🌐 Opens websites using voice commands
* 🤖 Basic conversational responses
* ⌨ Continuous listening until the user says **"bye"**

---

## 🛠 Technologies Used

* **Python**
* **SpeechRecognition** – to recognize voice commands
* **pyttsx3** – to convert text to speech
* **webbrowser** – to open websites
* **time** – to manage delays

---

## 📦 Required Libraries

Install the required libraries using pip:

```bash
pip install SpeechRecognition
pip install pyttsx3
pip install pyaudio
```

⚠️ Note: `pyaudio` is required for microphone input.

## 🎙 Example Commands

You can try the following voice commands:

| Command             | Action                     |
| ------------------- | -------------------------- |
| "Hello"             | Bot greets the user        |
| "How are you"       | Bot responds               |
| "Open YouTube"      | Opens YouTube in browser   |
| "Open Instagram"    | Opens Instagram in browser |
| "What is your name" | Bot introduces itself      |
| "Bye"               | Ends the program           |
